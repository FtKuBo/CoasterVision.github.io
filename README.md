# GNG2501 Project Website

## Project Overview
This website was developed as part of the GNG2501 course project. It serves as a platform to control our prototype of a hydraulic arm designed to assist with loading wheelchairs into vehicles. The website allows users to interact with the arm, adjusting its movement and monitoring its status in real-time. The arm is controlled using an ESP32 Arduino connected via Bluetooth.
